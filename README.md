# BeautyDiary Backend

Backend API for BeautyDiary, a beauty-service management application for salons and independent beauty professionals.

The backend is designed to support client records, appointment workflows, and service management features.

## Features

- Node.js backend structure
- API foundation for client and appointment data
- Database connection setup
- Modular server structure

## Tech Stack

- Node.js
- JavaScript
- Express
- Database connection via `db.js`

## Engineering Focus

- API structure for service-management workflows
- Separation of server and database logic
- Scalable backend foundation for future features
